# Tampere University IEM Citation Style

This repository contains a Zotero citation style specifically designed for students studying Industrial Engineering and Management (IEM) at Tampere University.

## Installation

To use this citation style, follow these steps:

1. Download the `tampere-university-iem.csl` file from this repository.
2. Open Zotero and go to `Edit` -> `Preferences`.
3. In the Preferences window, select `Cite` -> `Styles`.
4. Click on the `+` button to add a new style.
5. Choose the downloaded `tampere-university-iem.csl` file.

Now, you should be able to select the Tampere University IEM citation style when citing your references.

## Usage

1. Open Zotero and select the reference you want to cite.
2. Click on the `Cite` button.
3. Choose `Generate Bibliography` or `Insert Citation` as needed.
4. Select the Tampere University IEM citation style.
